JDK AND ITS VERSIONS

What is JDK ?

The java Development kit(JDK) is a distribution of Java Technology by Oracle corporation .It implements the Java Language specification(JLS) and the Java Virtual Machine specification (JVMS)and provides the Standard Edition(SE) of the Java Application Programming interface(API). It is a development environment for building applications,applets,and components using Java Programming Language and running on the Java Platform. JDK is a Software development environment that offers a collection of tools and libraries necessary for developing Java applications. The Oracle corporation has decided to release new JDK version for every 6 months with enhancements and newly add-on features which acts as better tools for developing applications.
